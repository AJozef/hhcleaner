# hhcleaner

CLI для очистки hh.ru: удаляет отклики-отказы, чаты-отказы, чаты по архивным вакансиям и старые чаты. Личная утилита для тех, кто давно ищет работу через hh и устал вручную разгребать накопившиеся отказы.

Неофициальный инструмент. С hh.ru никак не аффилирован.

> ⚠️ Платформа: только Windows. Скрипт использует Windows Task Scheduler и WinRT-уведомления; macOS/Linux не поддерживаются.

## Установка

```powershell
git clone https://github.com/AJozef/hhcleaner.git
cd hhcleaner
python -m venv .venv
.venv\Scripts\activate
pip install -e .
hhcleaner --setup
```

`--setup` скачает Chromium для Playwright (один раз, ~150 МБ) и откроет окно входа в hh.ru. Капчу и 2FA проходите вручную; после успешного входа сессия сохранится в `~/.hhcleaner/userdata/` и будет переиспользоваться при всех последующих запусках.

## Использование

```powershell
hhcleaner                            # все шаги по умолчанию
hhcleaner negotiations               # только отклики-отказы
hhcleaner old-chats --days 30        # только чаты старше 30 дней
hhcleaner --dry-run                  # показать что будет удалено, ничего не трогая
hhcleaner --status                   # статистика по чатам без удаления
hhcleaner --check                    # проверить, рабочая ли сессия
hhcleaner --self-check               # диагностика окружения
hhcleaner --install-schedule         # зарегистрировать еженедельный запуск
hhcleaner --help                     # все опции
```

Доступные шаги: `read-all`, `negotiations`, `chats-rejected`, `archived-vacancy`, `old-chats`, `chats-rejected-browser`.

## .env (опционально)

Если положить рядом файл `.env` с `HH_EMAIL` и `HH_PASSWORD`, скрипт будет:
- автоматически заполнять форму при ручном входе (капчу/2FA всё равно нужно пройти руками);
- в плановых `--no-input` прогонах при истёкшей сессии пытаться сделать **тихий headless-перелогин** — без открытия окна. Если в этот момент hh.ru покажет капчу или 2FA, перелогин не справится и придёт системное уведомление «нужен ручной вход».

Подробнее в [.env.example](.env.example).
