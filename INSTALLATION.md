# Установка и первый запуск hhcleaner

Подробная инструкция для обычного пользователя. Если коротко: скачать один файл и дважды кликнуть.

## Шаг 1. Скачайте программу

Откройте [страницу релизов](https://github.com/AJozef/hhcleaner/releases/latest) и скачайте файл **`hhcleaner.exe`** (~50 МБ). Это один файл, никакой установки и Python не требуется.

## Шаг 2. Запустите двойным кликом

Дважды кликните по `hhcleaner.exe`. Откроется окно, которое всё объяснит и проведёт по шагам.

### Если появилось синее окно «Windows защитила ваш компьютер» (SmartScreen)

Это стандартное предупреждение Windows для новых программ, которые ещё не подписаны платным сертификатом. Программа от этого не становится опасной — исходный код открыт, можете проверить.

Чтобы запустить: нажмите **«Подробнее»**, затем **«Выполнить в любом случае»**.

## Шаг 3. Войдите на hh.ru

При первом запуске откроется окно браузера с обычной страницей входа hh.ru. Войдите как всегда: логин, пароль, при необходимости — капча и код из почты/SMS (их вводите сами, вручную).

**Важно про безопасность:** вход происходит на настоящем сайте hh.ru в окне браузера. Программа **не видит и не сохраняет ваш пароль**. На компьютере остаётся только cookie сессии (то же самое, что остаётся, когда вы просто залогинены в браузере). Всё лежит локально, в папке `~/.hhcleaner/`, и никуда не передаётся.

После входа программа сама почистит аккаунт, покажет, что удалила, и подождёт, пока вы нажмёте Enter, — окно не закроется само.

## Готово

Дальше просто запускайте `hhcleaner.exe` двойным кликом, когда захотите почистить аккаунт. Повторный вход не нужен, пока сессия жива.

## Автозапуск по расписанию (необязательно)

Самый простой способ — ничего не делать руками: в конце работы визард сам спросит «Настроить автоочистку раз в неделю?». Ответьте `y` — и программа будет чистить аккаунт сама (по умолчанию по понедельникам в 9:00), без открытия окна.

Если предпочитаете настроить вручную или поменять день/время — откройте PowerShell в папке с `hhcleaner.exe` и выполните:

```powershell
# Еженедельный запуск (по умолчанию — понедельник, 9:00)
.\hhcleaner.exe --install-schedule

# Свой день и время:
.\hhcleaner.exe --install-schedule --schedule-day FRI --schedule-time 10:00

# Убрать из планировщика:
.\hhcleaner.exe --uninstall-schedule
```

Плановый запуск идёт без окна. Если сессия истекла, придёт системное уведомление с просьбой войти заново.

## ⚙️ Файл .env (необязательно)

Рядом с `hhcleaner.exe` можно положить файл `.env`:

```env
HH_EMAIL=your.email@example.com
HH_PASSWORD=your_password
```

Это позволит автоматически заполнять форму входа (капчу/2FA всё равно нужно пройти вручную) и пытаться тихо перелогиниться в плановых прогонах. Не обязательно и не коммитьте этот файл никуда.

## ❓ Частые вопросы

**Это безопасно? Куда уходят мои данные?**
Никуда. Все запросы идут напрямую с вашего компьютера на hh.ru, без промежуточных серверов. Пароль не сохраняется, на диске — только cookie сессии в `~/.hhcleaner/`. Исходный код открыт на GitHub.

**Нужно ли ставить браузер?**
Нет. Используется браузер, который уже есть в системе: Microsoft Edge (он встроен в Windows 10/11) или Google Chrome.

**Почему .exe пару секунд «думает» при запуске?**
Он распаковывает себя во временную папку (~2–3 сек). Это нормально, дальше быстрее.

**Сколько идёт очистка?**
Зависит от числа чатов: ~1 минута на сотню, ~10 минут на тысячу (между запросами есть паузы, чтобы hh не ругался). Скорость/объём можно ограничить флагами `--max-delete` и посмотреть заранее через `--dry-run`.

**Можно на Linux/macOS?**
Готового .exe — нет (планировщик и уведомления завязаны на Windows). Из исходников CLI запустится, но без планировщика и toast-уведомлений.

**Как обновиться?**
Скачать новый `hhcleaner.exe` из [Releases](https://github.com/AJozef/hhcleaner/releases/latest) и заменить старый.

## 📞 Если что-то не работает

Откройте PowerShell в папке с программой и запустите диагностику:

```powershell
.\hhcleaner.exe --self-check
```

Она проверит версию, наличие браузера для входа, сессию, каталог данных и лог и подскажет, что не так.
